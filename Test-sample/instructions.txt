1.This How to check your onion address--->>
sudo cat /var/lib/tor/securechat/hostname

2.Fast connection procedure
(For host)
cd ~/Tor_Anonychat
python3 -m securechat host --port 57311

(Guest Mode)
cd ~/Tor_Anonychat
python3 -m securechat connect --port 57311

quick summary

In-chat commands
Once connected, you type messages normally. Commands start with /:
CommandWhat it does/helpShow all commands/clearClear the message history from screen/quit or /qClose the session cleanly/sendfile <path>Offer a file to your peer/acceptAccept an incoming file offer/rejectDecline an incoming file offer
Keyboard shortcuts:
KeyAction↑ / ↓Scroll message historyCtrl-WClear the input lineCtrl-CQuit

File transfer
Files are transferred in-band over the same encrypted Tor channel as your messages — no second port, no extra connections.
Sending a file:
/sendfile ~/documents/report.pdf
/sendfile /tmp/photo.jpg
Your peer will see:
📁  Peer wants to send: report.pdf  (1.2 MB)  — type /accept or /reject
After they /accept, you'll both see progress:
📤  report.pdf: 62%  756 KB/1.2 MB  @ 91.3 KB/s
✅  report.pdf sent successfully in 13.4s (89.6 KB/s avg)
